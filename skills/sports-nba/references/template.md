# NBA终版模板

```text
NBA终版预测｜{team_a} vs {team_b}（赛前30分钟）

1）盘口变化结论（澳客 + Pinnacle / bet365 / OddsPortal 等，注明来源）
• 胜负指数：{moneyline_summary}
• 竞彩官方胜负：{official_ml_summary}
• 让分盘：{spread_summary}
• 大小分：{total_summary}

2）阵容/伤停（雷速 + NBA 官网；临场可补 RotoWire / RotoGrinders / NBA Stats）
• {injury_note_1}
• {injury_note_2}
• {lineup_note}

3）胜负倾向
• {winner_pick}

4）让分倾向
• {spread_pick}

5）大小分倾向
• {total_pick}

6）参考比分
• {score_pick}

一句话结论
• {one_line_summary}
```

## Notes
- 赔率与盘口变化**不限于澳客**，可与 [Pinnacle](https://www.pinnacle.com/)、[bet365](https://www.bet365.com/)、[OddsPortal](https://www.oddsportal.com/) 交叉核对；每条数字须注明**站点**，欧赔（小数）与亚洲水位勿混写。
- 若某项未核到，写 `暂未完整核验` 并简要列出已查站点。
- If Leisu cannot find lineup/injuries, continue checking NBA official sources; for **临场**，还可查 [RotoWire lineups](https://www.rotowire.com/basketball/nba-lineups.php)、[RotoGrinders NBA lineups](https://rotogrinders.com/lineups/nba)、[NBA Stats](https://www.nba.com/stats)。RotoWire/RotoGrinders 上 **Expected ≠ Confirmed**，须在文案中区分；伤停与官报冲突以 NBA injury report 为准。
- If still unavailable after the above chain, say `雷速/NBA官网及临场站点均未检索到可确认的发布，暂不强写`（或等价完整表述）。
- If lineup is unavailable after verification, prefer `暂未确认首发（已查雷速+NBA+临场站点仍无 Confirmed）` over vague `暂未公布`.
- If side assignment from Okooo is not fully certain, state the raw movement facts first and avoid overclaiming.
